# 🌊 TideWise - Coastal Tide Level Prediction & Safety Advisory System

A Machine Learning-based Flask web application that predicts coastal tide levels (Low, Normal, or High) using environmental and lunar parameters and provides safety recommendations.

## Features

- Predicts tide levels using Machine Learning
- Compares multiple classification algorithms
- User-friendly Flask web interface
- Provides safety advisory based on prediction
- Displays prediction confidence

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS

## Machine Learning Models

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Random Forest
